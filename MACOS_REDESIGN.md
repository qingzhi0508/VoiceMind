## 🚀 使用流程

1. **启动应用** → 显示欢迎页面
2. **点击"开始使用"** → 进入权限检查
3. **授予必要权限** → 进入准备就绪页面
4. **点击"启动服务"** → 开始运行
5. **在 iPhone 上配对** → 开始使用

- `startNetworkServices()` - 启动网络服务

- `refreshPublishedState()` - 刷新发布状态
