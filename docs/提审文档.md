## 二、核心功能描述

### 2.1 一句话介绍

VoiceMind — 用 iPhone 麦克风在 Mac 上实现实时语音输入。

### 2.2 详细描述（App Store Listing）

```
VoiceMind 将您的 iPhone 转变为 Mac 的无线麦克风，让语音输入更加便捷高效。

★ 核心功能

• iPhone 麦克风 → Mac 实时显示识别结果
• 6 位数字配对码 + HMAC-SHA256 加密，隐私安全
• Apple Speech 本地语音识别，无需联网
• 支持中文（普通话）和英文
• Bonjour 自动发现，零配置连接

★ 使用场景

• 写文档、邮件、代码时用语音输入
• iPhone 放桌上，Mac 远程收音
• 避免 Mac 键盘噪音干扰录音

★ 隐私保证

• 语音数据不离开本地网络
• 不上传任何服务器
• 配对信息存储在系统 Keychain
• 每次会话使用独立加密密钥

★ 系统要求

• macOS 14.0+ / iOS 18.0+
• iPhone 与 Mac 需要在同一 Wi-Fi 网络

使用方法：
1. 在 Mac 和 iPhone 上分别启动 VoiceMind
2. 在 iPhone 上输入 Mac 显示的 6 位配对码
3. iPhone 开始说话，Mac 实时显示文字
```

## 四、隐私政策

### 4.1 隐私政策 URL（必须）

> 如无自有隐私政策页面，可使用以下模板地址（建议搭建真实页面）：
> `https://voicemind.top-list.top/privacy.html`

### 4.2 隐私清单（已集成）

`PrivacyInfo.xcprivacy` 配置内容：

| 字段 | 值 | 说明 |
|------|-----|------|
| NSPrivacyTracking | false | 不追踪用户 |
| NSPrivacyCollectedDataTypes | [] | 不收集任何数据 |
| NSPrivacyAccessedAPITypes | CA92.1 | 仅访问 UserDefaults |

### 4.3 权限使用情况

| 权限 | 用途 | 数据处理 |
|------|------|---------|
| NSMicrophoneUsageDescription | 采集语音 | 本地处理，不上传 |
| NSSpeechRecognitionUsageDescription | 语音转文字 | Apple Speech 本地框架 |
| NSLocalNetworkUsageDescription | 发现 Mac | 仅局域网内通信 |
| NSCameraUsageDescription（二维码配对） | 扫二维码 | 仅本地解析配对码 |

## 七、构建与上传流程

### 7.1 iOS 构建上传

```bash
# 1. 清理后重新构建 Release 版本
cd /Users/cayden/Data/my-data/voicemind
xcodebuild -workspace VoiceMind.xcworkspace \
  -scheme VoiceMindiOS \
  -configuration Release \
  clean archive \
  -archivePath build/VoiceMindiOS.xcarchive

# 2. 使用 Xcode Organizer 上传
open -a Xcode build/VoiceMindiOS.xcarchive
# Xcode → Origanizer → 选择归档 → Distribute → App Store Connect

# 3. 或使用命令行上传
xcodebuild -exportArchive \
  -archivePath build/VoiceMindiOS.xcarchive \
  -exportOptionsPlist exportOptions_iOS.plist \
  -exportPath build/output
```

### 7.2 macOS 构建上传

```bash
# 1. 构建 Release
xcodebuild -workspace VoiceMindMac.xcworkspace \
  -scheme VoiceMindMac \
  -configuration Release \
  clean archive \
  -archivePath build/VoiceMindMac.xcarchive

# 2. 上传到 App Store Connect
xcodebuild -exportArchive \
  -archivePath build/VoiceMindMac.xcarchive \
  -exportOptionsPlist exportOptions_macOS.plist \
  -exportPath build/output
```
