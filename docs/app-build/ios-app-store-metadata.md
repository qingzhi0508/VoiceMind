`语灵 是一款轻量的 iPhone 语音转文字工具。你可以随时在手机上进行语音输入，快速获得转写结果，用于记录灵感、整理内容或减少日常打字。`

`当你需要更高效的跨设备体验时，语灵 也可以与 Mac 配合使用，将语音转写结果同步到 Mac 中查看与整理。`

`你可以用 语灵：`

- `在 iPhone 上快速开始语音转文字`
- `随时记录灵感、短句和待办内容`
- `减少输入成本，让表达更自然`
- `与 Mac 配对后同步语音结果，提升跨设备工作效率`

`主要特性：`

- `轻量直接的 iPhone 语音转文字体验`
- `适合记录、输入与轻量生产力场景`
- `支持与 Mac 协作同步语音结果`
- `帮助你减少打字，提升记录效率`

`语灵 需要以下权限以提供完整功能：`

- `麦克风权限：用于采集语音输入`
- `语音识别权限：用于将语音转换为文本`
- `本地网络权限：用于发现并连接同一局域网中的 Mac 设备`

## Review Notes

Use the English version below in App Store Connect review notes:

`Yuling (语灵) is a lightweight iPhone voice-to-text utility.`

`The iPhone app supports two usage modes:`

- `Standalone speech-to-text use directly on iPhone`
- `Optional collaboration with the companion macOS app over the local network`

`How to review:`

`1. Launch the iPhone app and allow microphone, speech recognition, and local network access if prompted.`
`2. You can test the app as a standalone voice-to-text tool directly on iPhone.`
`3. If testing the collaboration flow, launch the companion macOS app on the same local network.`
`4. Pair the devices using the in-app pairing flow.`
`5. Speak on iPhone and confirm the recognized text appears in the iPhone app and can also be received by the macOS app after pairing.`

`Permissions used:`

- `Microphone: used to capture speech input on iPhone`
- `Speech Recognition: used to convert speech into text`
- `Local Network: used for Bonjour discovery and communication with the companion macOS app on the same Wi-Fi network`

`Important clarifications:`

- `The iPhone app can be used independently as a speech-to-text tool`
- `Mac collaboration is optional and only used when both devices are on the same local network`
- `The app does not require account login to test core functionality`
