# 模型管理使用指南

## 概述

VoiceMind Mac 端提供了完整的模型管理功能，允许用户从网上下载语音识别模型并设置为默认使用。本指南将详细介绍如何使用这些功能。

## 模型管理界面

在 VoiceMind 的主界面中，点击 "语音识别" 选项卡，您将看到模型管理部分。

### 界面元素

- **模型列表**：显示所有可用的模型
- **模型信息**：包括模型名称、描述、引擎类型、大小和支持的语言
- **下载按钮**：点击下载未下载的模型
- **删除按钮**：删除已下载的模型
- **设为默认按钮**：将模型设置为默认使用
- **下载进度条**：显示模型下载进度

## 下载模型

1. 在模型管理部分找到您需要的模型
2. 点击 "下载" 按钮
3. 等待下载完成，进度条会显示下载进度
4. 下载完成后，模型状态会变为 "已下载"

### 支持的模型

- **SenseVoice Small**：多语言语音识别模型，支持50+语言，大小约85MB

## 设置默认模型

1. 确保模型已下载完成
2. 点击模型旁边的 "设为默认" 按钮
3. 模型会显示 "默认" 标记，表示已设置为默认模型

## 删除模型

1. 找到已下载的模型
2. 点击 "删除" 按钮
3. 模型会被删除，状态变为未下载

## 模型存储位置

模型默认存储在以下位置：

```
~/Library/Application Support/VoiceMindMac/Models/
```

每个模型会存储在以模型ID命名的子目录中。

## 故障排除

### 下载失败

- 检查网络连接
- 确保有足够的磁盘空间
- 检查防火墙设置

### 模型加载失败

- 确保模型下载完整
- 检查模型文件是否损坏
- 重启应用后重试

### 引擎不可用

- 确保模型已下载
- 检查模型文件是否完整
- 尝试重新下载模型

## 常见问题

**Q: 下载模型需要多长时间？**
A: 下载时间取决于网络速度，通常需要1-5分钟。

**Q: 可以同时下载多个模型吗？**
A: 目前只支持一次下载一个模型。

**Q: 模型下载后会占用多少磁盘空间？**
A: 不同模型大小不同，SenseVoice Small 约占用85MB空间。

**Q: 可以使用自己的模型吗？**
A: 目前只支持内置的预定义模型。

**Q: 模型更新后需要重新下载吗？**
A: 是的，当模型有新版本时，需要重新下载。
